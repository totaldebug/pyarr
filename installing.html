Installation

Install the package (or add it to your requirements.txt file):

poetry add pyarr

pip install pyarr

from source:

pip install -e https://github.com/totaldebug/pyarr.git#egg=pyarr

add this to requirements.txt:

-e git+https://github.com/totaldebug/pyarr.git#egg=pyarr

Via Git or Download

  1. Go to Pyarr Repo

  2. Download a copy to your project folders

  3. Import as below

from pyarr import Sonarr, Radarr, Readarr, Lidarr, Prowlarr, Bazarr, Whisparr, Dispatcharr
